## 简介

`EagleAdmin Vue` 3.0

## 安装使用

- 获取代码

```
git clone https://github.com/huguijian/eagleadmin-vue.git
```

- 安装依赖

```
cd eagleadmin-vue && yarn install
```

- 运行

```
yarn dev
```

- 打包

```
yarn build
```
